Explanination on CSS

CSS stands for cascading style sheets.It was developed to seprate the design of a webpage from its content.And the understanding of the document structure and the relationship between element is centarl to your work as a style sheet author.This makes website easier to design and maintain.

Common CSS proprties

property Function
clour Changes text color
background-color changes background color
font-size Changes text size
margin adds space ouside an element
padding adds space outside an element
