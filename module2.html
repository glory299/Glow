Introduction to HTML

HTML ia a markup language used to stucture web pages. It is neither compiled nor interpreted like other languages but rendered directly by the browsers,displaying contect without typical errors, thereby making execution smooth.

HTML tags and attributes

HTML tags are special keywords enclosed in angle brackets.They are opened and closed bracket with forward slash.Attributes provides additonal information about HTML elements.for example, the "src" attributes specifies the source of an image.

HTML page structure

An HTML page structure contains the DOCTYPE declaration,html tag.head section and body section. These element form the basic structure of every webpage.And non of the above mentioned can be negleted to above errors.
