UNIT 1:DEFINITIONS OF WEB PROGRAMMING,INTERNET AND WWW

WEB PROGRAMMING:is the process of writing codes that will eventually creates a websites.Texts and multimedia contents can be manipulated by a web designer to produce an elegant page that is seen on a web browser.Web programming is therefore the creation of web pages that run over the internet. INTERNET:the internet is a global communication system consisting of hardware and softwear infrastucture that provides connectivity between remoye computer. THE WORLD WIDE WEB(WWW):it is a hypermedia system.Invented by TIM BERNERS-LEE in 1994, its a global connection of interconected documenton the internet.The www is a part of the internet that uses hypertext protocol(HTTP) to display hypertext and images in agraphical environment.It is a hypertext based information retrival tool.
